## Tier 1 — Google Search Console (Do This First, Today)

**Why it matters:** Without submitting your sitemap, Google discovers your pages by crawling — which can take weeks or months. Submitting directly tells Google to index all 186 pages immediately.

**Steps:**
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add property: `https://www.alderheritagehomes.com` (choose URL prefix)
3. Verify ownership via the HTML tag method — add the `<meta name="google-site-verification" ...>` tag to the site's `<head>` (send the tag value and it can be added in minutes)
4. Go to **Sitemaps** → Submit `https://www.alderheritagehomes.com/sitemap.xml`
5. Check **Coverage** weekly — fix any "Not indexed" errors

**Expected result:** All 186 pages indexed within 1–2 weeks instead of 2–3 months.

## Conversion Rate Optimization (Get More Leads from Existing Traffic)

More traffic is only valuable if it converts. These changes can increase lead volume without increasing traffic:

1. **Add a sticky "Get My Offer" bar** on all blog posts — a fixed bottom bar with a phone number and a one-click offer request button
2. **Add exit-intent popup** on city pages — trigger when the user moves to close the tab, offering a "Wait — get your free offer before you go" message
3. **Add social proof counters** — "47 homeowners requested an offer this month" on the homepage hero
4. **Speed up the form** — reduce the contact form to 3 fields (address, phone, timeline) and add the remaining fields on a second step
5. **Add a "What's My Home Worth?" tool** — a simple address lookup that captures leads while providing perceived value
