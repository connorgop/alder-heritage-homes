## Setup Steps in Google Ads

### Step 1: Create Campaign 1 (Branded Keywords)
1. Go to Google Ads (ads.google.com)
2. Click "Campaigns" → "New Campaign"
3. Campaign type: Search
4. Campaign name: "Alder Heritage - Branded Keywords"
5. Budget: $11.43/day
6. Networks: Search only (uncheck Display)
7. Location: California (or Fresno County)
8. Language: English
9. Bidding: Manual CPC
10. Create ad group "GREimagined"
11. Add keywords: greimagined, greimagined fresno, greimagined reviews
12. Max CPC: $7
13. Create ads (see copy above)

### Step 4: Set Up Conversion Tracking
1. Go to Google Ads
2. Click "Tools & Settings" → "Conversions"
3. Click "New Conversion Action"
4. Select "Website"
5. Enter your website: alderheritagehomes.com
6. Conversion type: "Phone calls" (track calls to 559-281-8016)
7. Get tracking code and add to website
8. Test conversion tracking

### Step 5: Link Google Analytics
1. Go to Google Ads
2. Click "Tools & Settings" → "Linked Accounts"
3. Click "Google Analytics"
4. Link your Google Analytics account
5. This will track traffic and conversions

## Optimization Tips

### If CTR is Low (<5%):
1. Rewrite headlines to be more compelling
2. Add numbers or power words
3. Make it clear what the page is about
4. Test different ad variations

### If CPC is High (>$8):
1. Improve Quality Score (better landing page)
2. Increase bids on high-converting keywords
3. Decrease bids on low-converting keywords
4. Pause underperforming ads

### If Conversion Rate is Low (<2%):
1. Check landing page relevance
2. Improve page speed
3. Add clear CTAs
4. Test different landing pages

### If CPA is High (>$50):
1. Improve ad targeting
2. Pause underperforming keywords
3. Increase bids on high-converting keywords
3. Test new ad copy
