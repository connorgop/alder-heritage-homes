## Option A — "Warm Authority" (American Craftsman + Trust-First)

**Design Movement:** American Craftsman meets modern editorial real estate

**Core Principles:**
1. Warmth and approachability — distressed sellers are scared; the site must feel like a trusted neighbor, not a predator
2. Authority through credentials — DRE license, 100+ sales, testimonials front and center
3. Clarity over cleverness — every headline answers a real question the seller is asking
4. Asymmetric editorial layouts — content-heavy but never overwhelming

**Color Philosophy:**
- Deep forest green (`#1C3A2A`) as primary — evokes stability, heritage, nature, trust
- Warm cream/parchment (`#F5EDD8`) as background — not stark white, feels like a letter from a trusted advisor
- Amber gold (`#C8860A`) as accent — warmth, value, action
- Charcoal (`#2D2D2D`) for body text

**Layout Paradigm:**
- Full-bleed hero with angled diagonal cut between sections
- Left-heavy editorial layout with large pull quotes
- Sticky sidebar navigation on service pages
- Card-based "situation selector" on homepage (I'm facing foreclosure / I inherited a home / etc.)

**Signature Elements:**
- Alder leaf motif (subtle, in dividers and section backgrounds)
- Hand-drawn underline accents on key phrases
- "Situation cards" — large illustrated cards for each seller scenario

**Interaction Philosophy:**
- Scroll-triggered fade-ins for trust signals
- Hover states on cards reveal a reassuring sub-headline
- Phone number always visible in sticky header

**Animation:**
- Gentle fade-up on section entry (0.4s ease-out)
- Subtle parallax on hero background
- Number counters animate when scrolled into view (100+ sales, 24hr offers, etc.)

**Typography System:**
- Display: `Playfair Display` — authoritative, warm, editorial
- Body: `Source Sans 3` — readable, professional, accessible
- Accent labels: `DM Mono` — for stats, license numbers, phone

## Option C — "Heritage Warmth" (Organic + Earthy Editorial) ← SELECTED

**Design Movement:** Pacific Coast Heritage meets ethical real estate editorial

**Core Principles:**
1. Human-first storytelling — the site speaks TO the seller's situation, not AT them
2. Earthy warmth — colors and textures evoke California soil, oak trees, and community
3. Generous whitespace — breathing room signals confidence and respect
4. Layered credibility — credentials woven naturally into narrative, not plastered

**Color Philosophy:**
- Warm terracotta (`#B5541C`) as primary — California earth, warmth, action
- Deep slate green (`#1E3A2F`) as secondary — heritage, stability, nature
- Oat/linen (`#F7F2E8`) as background — organic, warm, not clinical
- Warm charcoal (`#2C2C2C`) for body text
- Soft sage (`#8FAF8A`) as a supporting accent

**Layout Paradigm:**
- Asymmetric hero: large left text block, right-side image with organic shape mask
- "Situation Finder" section: horizontal scrolling cards for each seller scenario
- Diagonal section dividers (CSS clip-path) between major sections
- Full-bleed testimonial sections with large background text
- Blog uses a magazine-style 2+1 grid

**Signature Elements:**
- Alder leaf watermark in section backgrounds (very subtle, 4% opacity)
- Organic blob/shield shape masking hero images
- Warm gradient overlays on photo sections

**Interaction Philosophy:**
- Situation cards expand on hover to show a brief description + CTA
- Scroll-triggered counter animations for key stats
- Phone number pulses gently in header on mobile

**Animation:**
- Staggered fade-up for hero elements (0.3s, 0.5s, 0.7s delays)
- Situation cards: scale 1.02 on hover with shadow deepening
- Section entry: translateY(24px) → 0 with opacity 0 → 1 over 0.5s

**Typography System:**
- Display: `Lora` (serif) — warm, trustworthy, editorial authority
- Body: `Nunito Sans` — friendly, readable, approachable
- Stats/Labels: `DM Mono` — grounded, precise, credible
- Font sizes: Display 56-72px, H2 36-42px, H3 24px, Body 17px/1.7 line-height
