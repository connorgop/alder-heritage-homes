## Part 1: Create Retargeting Audiences

### Step 1.1: Go to Google Ads Audiences

1. Open Google Ads: https://ads.google.com
2. Click **Tools & settings** (wrench icon) in top right
3. Click **Audiences** under "Shared library"
4. Click **Create audience** (blue button)

**You should see:**
```
Create audience
├── Website visitors
├── App users
├── YouTube users
├── Customer match
└── Similar audiences
```

### Step 1.2: Create Audience #1 - All Website Visitors

**Click:** Website visitors

**Fill in these fields:**

| Field | Value |
|-------|-------|
| Audience name | All Website Visitors - 30 days |
| Membership duration | 30 days |
| Website visitors | All visitors |

**Click:** Create audience

**Result:** You now have Audience #1 tracking all visitors for 30 days.

---

### Step 1.3: Create Audience #2 - Lead Capture Page Visitors

**Click:** Create audience > Website visitors

**Fill in these fields:**

| Field | Value |
|-------|-------|
| Audience name | Lead Capture Page Visitors - 7 days |
| Membership duration | 7 days |
| Website visitors | People who visited specific pages |
| URL contains | /ads/lead-capture |

**Click:** Create audience

**Result:** You now have Audience #2 tracking only people who visited your lead capture form.

---

### Step 1.4: Create Audience #3 - Blog Readers

**Click:** Create audience > Website visitors

**Fill in these fields:**

| Field | Value |
|-------|-------|
| Audience name | Blog Readers - 14 days |
| Membership duration | 14 days |
| Website visitors | People who visited specific pages |
| URL contains | /blog |

**Click:** Create audience

**Result:** You now have all 3 audiences created!

## Troubleshooting

### Problem: Campaigns show "Limited by budget"
**Solution:** Increase daily budget or reduce Target CPA bid

### Problem: No impressions after 48 hours
**Solution:** 
1. Check audience size (need 100+ people minimum)
2. Check bid strategy (may be too low)
3. Check location targeting (make sure Fresno is included)

### Problem: High cost per conversion ($50+)
**Solution:**
1. Lower Target CPA bid
2. Improve landing page (reduce form fields)
3. Test new ad copy

### Problem: Audience size shows 0
**Solution:**
1. Wait 24-48 hours (audiences take time to build)
2. Check URL filters are correct
3. Verify conversion tracking is working
